# Github Declutter

Declutter your unused repo. Create your github personal access token, give permission for these scopes:

* repo
* gist
* delete_repo

then declutter your repo.

![docs/img/screenshot.png](docs/img/screenshot.png)

## Deveelopment

```sh
$ npm install
$ npm run dev -- --host
```
